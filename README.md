# Dog Behaviourist Melbourne

This is the official website for Dog Behaviourist Melbourne, a service connecting dog owners with professional behaviorists in the Melbourne area.

## Development

The site uses:
- HTML5, CSS3, and JavaScript
- Bootstrap 5 for responsive design
- Google Maps JavaScript API for location services

## Google Maps API Key

To make the map functionality work:
1. Obtain a Google Maps JavaScript API key from the [Google Cloud Console](https://console.cloud.google.com/)
2. Replace the placeholder `YOUR_GOOGLE_MAPS_API_KEY_HERE` in the index.html file with your actual API key

## Deployment

This site is hosted on GitHub Pages at https://dogbehaviouristmelbourne.github.io/
